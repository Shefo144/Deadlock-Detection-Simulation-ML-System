# 🔥 Deadlock Prediction & Simulation System

## 📌 Project Overview
This project focuses on predicting whether a deadlock will occur in an Operating System based on user input data.  
It also includes a graphical simulation to visualize how deadlock happens in real-time using animation and graphics.

The system integrates:
- Operating Systems concepts (Deadlock)
- Machine Learning models
- Data Visualization and Simulation

---

## 🎯 Objective
- Predict if a deadlock will occur based on system input
- Compare multiple Machine Learning models
- Select the best performing model
- Visualize deadlock behavior using simulation

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## 🤖 Machine Learning Models Used
- Logistic Regression  
- Support Vector Machine (SVM)  
- Random Forest  

The best model was selected based on accuracy and performance.

---

## 📊 Workflow
1. Data Input from user
2. Data Preprocessing
3. Feature Selection
4. Model Training
5. Model Comparison
6. Prediction (Deadlock or No Deadlock)
7. Graphical Simulation

---

## 📈 Results
- Multiple models were tested
- Best model achieved highest accuracy
- Output includes probability of deadlock occurrence

---

## 🎮 Features
- User input-based prediction
- Machine Learning classification
- Deadlock simulation and animation
- Visualization of system behavior
